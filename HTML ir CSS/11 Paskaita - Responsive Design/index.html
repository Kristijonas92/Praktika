Mobile First

  • Is mazesnio ekrano dydzio yra lengvesnis perejimas i didesni. (lengviau prideti naujo turinio, negu atimti seno)
  • Koncentracija i turini (del mazo vietos kieki)
  • Didesnis kiekis vartotoju naudoja mobiliasias aplikacijas. (orientojames i didesni vartotoju kieki)
  • Google paieskos algoritmas geriau ivertina Mobile First dizainu kurtas svetaines.
