Stilizavimo tag'ai

  • b - bold - Pastorina tekstą
  • Strong - Pastorina tekstą
  • i - Italic - Suteikia tekstui pasvyrimo stilių
  • em - emphasis - Suteikia tekstui pasvyrimo stilių (Svarbu semantiškai)
  • small - Sumažina tekstą
  • big - Padidina tekstą (Pasenęs - nenaudojamas)
  • sup - superscript - Pakelia tekstą į viršų ir jį sumažina
  • sub - subscript - Nuleidžia tekstą į apačią ir jį sumažina
  • Spun - išskiria kažkokį elemantą ar jo dalį

Formos

Tag'ai

  • Form - Forma
  • input - Įvestis
  • select - Pasirinkimas iš variantų
  • option - Pasirinkimo variantai
  • Textarea - didelio teksto kiekio
  • Label - Etiketės prie įvesties
  • optgroup - Pasirinkimo variantų grupė
  • button - Mygtukas
  • fieldset - Formos laukas
  • legend - Formos lauko pavadinimas

Atributai

  • Type - Input tag'o tipas
  • for - Nurodo kuriam įvesties/pasirinkties laukui yra priskirta etiketė
  • id - Unikalus pavadinimas, į kurį nurodo for atributas
  • name - Įvesties lauko vardas, kuris bus perduotas back'ui kartu su tos įvesties reikšme
  • cols - Stulpelių kiekis textarea elemente
  • rows - Eilučių kiekis textarea elemente
  • placeholder - Teksta esantis įvesties lauke, kuris prandingsta pradedant pildyti įvesties lauką
  • value - Įvesties reikšmė, kuri perduodama į back'ą
  • label - priskiria
  • select - Numatytasis select tag'o viduje esantis option'as
  • minlenght - Trumpiausia galima įvestis
  • maxlenght - Ilgiausia galima įvestis
  • min - Mažiausia leistina įvestis
  • max - Didžiausia įvestis
  • step - Žingsnis, kuriuo didėja/mažėja skaičius
  • pattent - Reikalauja, kad įvesties laukelyje būtų parašytas nurodytas formatas (formatas nurodomas regex(regular expresion))
  • checked - Numatytasis radio arba checkbox pasirinkimas
